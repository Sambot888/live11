# 币安西游（Binance Odyssey）链游概念案

## 1. 项目定位
- **类型**：回合制社交 RPG + 轻策略经济模拟（“梦幻西游”式组队刷本 + 链上资产成长）。
- **世界观关键词**：神话修真、链上秩序、跨链乱世、门派博弈。
- **目标用户**：
  1. 喜欢国风神话和养成玩法的 RPG 玩家；
  2. 有钱包基础、但希望“先玩后懂 Web3”的泛用户；
  3. 热衷组队、公会、交易和赛季竞争的核心玩家。

## 2. 核心卖点（先讲玩家能感知的）
1. **熟悉感**：场景、美术和战斗节奏接近传统国风回合制。
2. **真资产**：角色皮肤、坐骑、法宝、家园部件可链上确权与交易。
3. **低门槛**：游客模式可直接玩，达到关键等级再引导绑定钱包。
4. **高社交**：五人组队副本、师徒系统、帮派城战、跨服天梯。
5. **动态剧情**：赛季剧情由链上事件触发（例如“某门派占据灵脉”）。

## 3. 世界观与主线故事
三界本有“灵脉天网”，维持人、仙、妖平衡。某日，来自“虚空链域”的混沌协议侵蚀天网节点，导致灵气失衡、门派法则崩塌。天庭发布“封链榜”，招募三界修士重铸节点。

玩家作为“新晋链使”，将经历三个章节：
- **第一章：断链长安**：调查长安灵脉断裂，建立初始队伍。
- **第二章：跨链东海**：前往东海龙宫，修复跨海传送阵并解锁跨服交易港。
- **第三章：天宫共识战**：门派与帮会争夺“共识权重”，决定赛季结局。

> 叙事重点：把 Web3 概念（节点、共识、跨链、治理）翻译为神话语境（灵脉、天书、传送阵、天命议会），让玩家“先代入，再理解”。

## 4. 关键场景设计（可直接用于立项分镜）

### 4.1 长安主城（新手与交易枢纽）
- 功能：主线接取、摆摊、拍卖行、师徒大厅、公会招募。
- 氛围：市井烟火 + 法阵全息屏（展示链上公告）。
- Web3 映射：
  - 拍卖行 = 链上市场入口；
  - 公告屏 = 链上事件看板（Boss 被击杀、稀有道具铸造）。

### 4.2 花果山（成长与资源采集）
- 功能：日常刷怪、采集、坐骑驯养、单人试炼。
- 差异化：地图事件随“灵气指数”变化，激励分时在线。
- Web3 映射：资源产出受赛季状态和公会控制加成影响（透明可查）。

### 4.3 东海龙宫（跨服与跨链港口）
- 功能：跨服副本入口、跨区交易、海域 Boss。
- 叙事：龙宫保管“万链潮汐盘”，负责各界资产通航。
- Web3 映射：
  - 跨服交易 = 跨链桥概念可视化；
  - 潮汐盘稳定度 = 系统流动性/手续费调节剧情化表现。

### 4.4 火焰山（高风险高收益 PVE）
- 功能：高难秘境、装备材料掉落、赛季排行榜。
- 机制：可选择“稳妥路线”或“业火捷径”（高爆率高翻车率）。
- Web3 映射：收益—风险透明披露，减少“黑箱感”。

### 4.5 天宫议会（治理与赛季决策）
- 功能：门派提案、帮会联盟、赛季规则投票。
- 叙事：玉帝授权“天命议会”，由各派代表共治三界。
- Web3 映射：轻治理系统，避免复杂 DAO 门槛；采用任务贡献 + 持有凭证双权重。

## 5. 人物体系（Web3 化角色模板）

## 5.1 主角阵营（玩家可结识/招募）
1. **链机子（军师）**
   - 原型：诸葛系智者 + 协议工程师。
   - 设定：擅长修补灵脉节点，负责新手链上教学。
2. **白骨夫人·重铸形态（刺客）**
   - 设定：曾被误判封印，渴望通过“身份凭证”重获名誉。
   - 玩法：高爆发、伪装、偷取 Boss 增益。
3. **龙太子·潮汐守望者（坦克）**
   - 设定：守护跨海资产航路，反对无序掠夺。
   - 玩法：护盾、嘲讽、团队减伤。
4. **玄奘·行愿共识者（辅助）**
   - 设定：传播“可信协作”的修行理念。
   - 玩法：治疗、净化、复活。

### 5.2 反派阵营
1. **无相魔主（主反派）**：以“无限增发”诱惑凡人，制造经济失衡。
2. **黑市判官（中层反派）**：操纵假道具与假凭证，破坏市场信任。
3. **噬链妖王（世界 Boss）**：吞噬跨链节点，周期性入侵各地图。

## 6. 玩法循环（玩家每天在做什么）
1. **15 分钟**：日常任务 + 单人速刷（保底收益）。
2. **30 分钟**：组队副本 + 门派任务（核心成长）。
3. **弹性时段**：摆摊交易、家园装修、帮派活动。
4. **周目标**：挑战赛季 Boss、冲击天梯、参与议会提案。

> 设计原则：PVE 获取大部分养成材料，PVP 主要给荣誉与外观，降低“纯数值内卷”。

## 7. 资产与经济模型（简化版）
- **双代币（建议）**
  - `YAO`（治理/高价值）：总量有限，主要来自赛季、贡献、活动。
  - `LING`（游戏内循环）：用于强化、打造、修理、传送。
- **资产分类**
  1. 非 NFT：常规材料、低阶道具（降低链成本）。
  2. 可上链 NFT：稀有装备、限定皮肤、传奇坐骑、地块装饰。
- **经济安全阀**
  - 材料回收炉（销毁过剩道具）；
  - 装备耐久与修复消耗；
  - 赛季重置部分排行榜奖励，避免通胀固化。

## 8. 新手引导（降低 Web3 门槛）
1. 第 1 天：游客登录，体验剧情与战斗。
2. 第 2 天：达成等级后赠送“灵契钱包”（可托管）。
3. 第 3 天：完成首次链上铸造（低价值纪念徽章）。
4. 第 4~7 天：通过任务学会交易、上架、撤单、资产保护。

## 9. 商业化建议（不破坏公平）
- 可售卖：外观、特效、坐骑皮肤、家园主题包、赛季通行证。
- 谨慎售卖：成长加速道具（设置上限，不影响竞技公平）。
- 禁止方向：直接售卖顶级战力装备。

## 10. 版本路线图（前 6 个月）
- **M1-M2（可玩验证）**：长安 + 花果山 + 基础战斗 + 组队副本。
- **M3-M4（经济闭环）**：交易行、基础上链资产、赛季任务。
- **M5（社交竞争）**：帮派战、跨服天梯、龙宫跨区港口。
- **M6（治理试点）**：天宫议会开放轻治理投票。
