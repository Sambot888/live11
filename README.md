# 币安西游（Binance Odyssey）可运行原型

这是一个基于 `binance-westward-game-design.md` 的前端可运行原型（纯 HTML/CSS/JS，无需安装依赖）。

## 运行方式

在仓库根目录执行：

```bash
python3 -m http.server 8000
```

然后在浏览器访问：

- http://localhost:8000/

## 文件说明

- `binance-westward-game-design.md`：完整策划文档
- `index.html`：页面结构
- `styles.css`：视觉样式
- `app.js`：场景/人物/玩法等交互逻辑
